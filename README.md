# E2E1-TaskTracker
공유 일정,목표 관리 서비스

##기술 스택
- Front-end:
    - Bootstrap
    - ThymeLeaf
- Back-end:
    - Java 17
    - Spring Framework 3.x.x
        - Spring Boot
        - Spring Data JPA
        - Spring Security
            - Kakao 로그인
            - OAuth2
            - JWT 인증(option)
    - WebSocket (소켓 통신) (option)
    - MySQL (데이터베이스)
    - AWS (Amazon Web Services)
    - Docker
    - Redash
